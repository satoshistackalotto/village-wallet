üöÄ Getting Started

Village Wallet is a physical NFC card that stores cryptocurrency. It works like a tap-to-pay card, but for EVM compatible chains, Ethereum, Polygon, Arbitrum, Optimism, Base, BNB Chain, USDT, USDC, DAI . Each card contains an encrypted wallet that you control with a PIN.

Key features:

  • Tap your phone to check balance
  • Enter PIN to authorize payments
  • Works on multiple networks
  • Can import to MetaMask
  • Open-source security
  1. Get a Village Wallet card - Order from our website or Etsy
  2. Download the app - Available for Android
  3. Load funds - Send crypto to your card's address
  4. Tap to pay - Use the app to authorize payments

See our Setup Guide for detailed instructions.

You need:

  • A Village Wallet NFC card
  • Android phone with NFC
  • Village Wallet app (free download)
  • Your PIN (included with card)

Optional:

  • MetaMask or other wallet (for advanced features)

üõ†Ô∏è Troubleshooting

Try these steps:

  1. Enable NFC - Settings ‚Üí Connected devices ‚Üí NFC (turn ON)
  2. Remove phone case - Cases can block NFC
  3. Hold card steady - On the back center of phone for 3-5 seconds
  4. Try different position - NFC antenna location varies by phone
  5. Restart app - Close and reopen Village Wallet

Still not working?

  • Test with another NFC app (like "NFC Tools")
  • Try on a different phone
  • Contact support if card is defective

Check these common issues:

  • Wrong card - Make sure you're using the right card for that PIN
  • Typo - PINs are case-sensitive (though usually numeric)
  • Card serial - Verify the serial number matches

If none of these work, try using the decryption tool on our website to verify your PIN.

Common reasons transactions fail:

  • Insufficient funds - Don't forget about gas fees!
  • Network congestion - Try again in a few minutes
  • Gas price too low - App should handle this, but try different network
  • Internet connection - Need stable connection to broadcast

The app will show you exactly what went wrong in the error message.
